# Security Policy

## Supported Versions

RepoXray is early-stage software. Security fixes are provided for the latest
version on the default branch.

## Reporting A Vulnerability

Please do not open public issues for security vulnerabilities.

Report security concerns by emailing the project maintainers, or by using
GitHub private vulnerability reporting if it is enabled for the repository.

Include:

- Affected version or commit.
- Steps to reproduce.
- Impact and expected behavior.
- Any suggested fix, if known.

We will acknowledge valid reports and coordinate a fix before public disclosure.
